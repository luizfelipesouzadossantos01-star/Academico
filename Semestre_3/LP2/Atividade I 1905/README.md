1) O que é um paradigma de programação?
É uma abordagem fundamental que define como voce estrutura, resolve e organiza problemas usando codigo.

2) Quais os principais paradigmas de programação?
As linguagens de programação costumam adotar um ou mais paradigmas. Os principais são divididos em dois grandes grupos:
Paradigma Imperativo
Estruturado: O código é executado de forma sequencial, focado em funções e blocos de controle. Exemplo: C.
Paradigma Orientado a Objetos: O código é estruturado em torno de "objetos" reais ou abstratos que contêm dados e comportamentos. Exemplo: Java, Python.
Paradigma Declarativo
Funcional: Baseia-se na avaliação de funções matemáticas e evita a modificação de estados ou dados globais. Exemplos: Haskell, Clojure.
Lógico: Baseado em lógica formal. Você dá ao programa um conjunto de fatos e regras, e o computador deduz a resposta. Exemplo: Prolog, .

3) O que é programação orientada a objetos? E quais suas vantagens?
A Programação Orientada a Objetos (POO) é um paradigma que aproxima a estrutura do código do mundo real. Em vez de pensar no programa apenas como uma sequência de comandos que manipula dados isolados, você passa a enxergar o sistema como um conjunto de entidades independentes que são os objetos que conversam entre si.
Suas vantagens são reutilização de Código onde você pode criar uma classe uma única vez e instanciar (new) milhares de objetos baseados nela. O código fica dividido em partes menores e lógicas facilitando sua manutenção sem o risco de quebrar o resto do sistema.

4) O que é abstração?
A abstração é o processo mental de separar e selecionar apenas as características essenciais de um objeto do mundo real que são importantes para o contexto do seu sistema, descartando todo o resto que for irrelevante.

5) O que é uma classe?
Uma Classe é a estrutura, o molde ou a planta baixa que define as características e comportamentos que um objeto terá. Ela não é o objeto em si. Nela, você escreve as variáveis e as funções.

6) Como base nos conceitos identifique características e
comportamentos dos objetos.
Celular
Características: marca, modelo, armazenamento.
Comportamento: ligar/desligar, fazer ligação, tirar foto.

Cão
Características: raça, nome, idade.
Comportamento: latir, comer, dormir.

Professor
Características: nome, materia, matricula.
Comportamento: explicar, auxiliar, planejar aula.

Televisão
Características: marca, tamanho, modelo. 
Comportamento: mudar canal, aumentar volume.

Disquete
Características: capacidade, espaço ocupado.
Comportamento: gravar dados, ler dados, formatar.
