# Configuração do Ambiente de Desenvolvimento Java

## Este documento apresenta a validação do ambiente Java configurado para as práticas da disciplina.

### A) Verificação da Versão do Java

Para confirmar a instalação e o funcionamento da máquina virtual, foi executado o comando java -version no terminal. O printscreen abaixo exibe a saída do comando, comprovando que o sistema operacional está executando a versão ativa do OpenJDK 21.

Para mapear a arquitetura física do Java no sistema, foi executado o comando tree diretamente no diretório de instalação (/usr/lib/jvm/java-21-openjdk-amd64). O printscreen a seguir exibe a hierarquia principal (nível 1), destacando os diretórios estruturais do JDK, como bin e lib.
